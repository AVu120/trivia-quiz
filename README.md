## How to Run Application

### View/edit/fork CodeSandbox at: https://codesandbox.io/s/github/AVu120/trivia-quiz

## Application Set Up

### `git clone https://github.com/AVu120/trivia-quiz.git`

### `cd trivia-quiz`

### `npm i`

### `npm start`

Then open http://localhost:3000 in a browser.
